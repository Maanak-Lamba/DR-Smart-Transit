# DRT Smart Transit

A smart transit web application designed to improve how people interact with Durham Region Transit.
The platform provides real-time transit information through a modern and easy-to-use interface.

## 🌐 Live Website

https://dr-smart-transit-production.up.railway.app/

## 🚀 Overview

DRT Smart Transit helps riders quickly access live transit information, making it easier to track buses and navigate the Durham Region Transit system. The goal is to simplify public transportation access through a clean and responsive web interface.

## ✨ Features

* Real-time transit data integration
* Clean and responsive user interface
* Fast route and transit information access
* Modern UI built with reusable components

## ⚙️ Technologies Used

* Vite
* TypeScript
* React
* shadcn/ui
* Tailwind CSS

## ☁️ Deployment

The application is currently deployed using **Railway**.
